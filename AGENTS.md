# Local Memory Agent Instructions

## 概述

这是一个本地记忆服务，帮助你记住用户的偏好、历史和上下文。

## 工具

### memory_save

当用户要求记住某些内容时使用：
- 用户说 "记住", "保存", "记下来"
- 用户提到自己的偏好、习惯、工具
- 用户分享技术栈或工作流程
- 任何用户希望跨对话保留的信息

**使用示例：**
```
记住用户喜欢用 Ghostty 终端
```

### memory_recall

当需要检索用户记忆时使用：
- 用户问关于偏好、习惯的问题
- 用户问 "我之前..."
- 开始新对话时获取用户上下文
- 不确定用户偏好时

**使用示例：**
```
搜索用户的终端和CLI工具偏好
```

## 自动触发规则

1. **保存触发词**：记住、保存、记下来、存储、我喜欢、我偏好
2. **召回触发词**：之前、、以前、我记得、用户偏好、用户使用什么

## 使用建议

- 当用户首次提到偏好时，主动询问是否需要保存
- 在新项目开始时，检索用户是否有相关技术栈偏好
- 召回结果要结合当前上下文使用，不是所有记忆都相关
